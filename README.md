# 🎉 파티룸 · 펜션 게임 모음

[`https://naryeo628.github.io/party-games/`](https://naryeo628.github.io/party-games/)

파티룸·펜션에서 바로 쓰는 게임 **룰북 + 라이브 진행 도구**. 폰 하나로 끝. 빌드 없음(정적 HTML 1파일).

## 구성

**🎮 라이브 도구**
- 🎯 랜덤 뽑기 / 순서 — 당첨자 1명, 전체 순서 셔플, 팀 나누기
- 🎡 벌칙 룰렛 — 항목 입력 → 회전 룰렛
- ⚖️ 밸런스 게임 — VS 카드 탭 선택, 카테고리별(취향·음식/극한/연애/황당·초능력) 120문항. 🌶️매운맛(19+)은 별도 선택·기본 제외
- 🕵️ 마피아 역할 배정기 — 인원/마피아 수/의사·경찰 설정 → 폰 돌리며 비밀 확인(pass-the-phone)
- 🎭 라이어 게임 진행기 — 카테고리 선택 → 제시어·라이어 자동 비밀 배정
- 🃏 제시어/질문 덱 — 몸으로말해요·초성·텔레파시 랜덤 뽑기

**📖 게임 룰북** — 마피아, 라이어, 369, 초성, 밸런스, 몸으로말해요, 텔레파시, 눈치, 캐치마인드, 이미지 게임 (인원/시간/난이도 + 규칙 + 팁)

## 콘텐츠 추가

`index.html` 상단 데이터 배열만 수정:
- `GAMES` — 룰북 게임
- `LIAR_DECKS` / `GESTURE_DECKS` — 제시어 풀(카테고리별)
- `CHOSUNG_WORDS` / `BALANCE_Q` / `TELEPATHY_Q` — 단어·질문 풀
